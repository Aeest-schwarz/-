# -AI HR Coach 
AI HR Coach — это интерактивное веб-приложение на базе GigaChat и Streamlit, которое помогает кандидатам узнавать тонкости составления резюме, сопроводительных писем, готовиться к собеседованиям. Система сопоставляет ответ пользователя с реальной базой данных мнений десятков HR-специалистов.
